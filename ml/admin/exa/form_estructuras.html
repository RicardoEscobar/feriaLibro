Estructuras de control

Uso de if

Proporciona dos números enteros, elabora una aplicación en PHP que determine cuál de ellos es el mayor.

Uso de switch

Proporciona un número entero que represente una calificación del 1 al 10. Elabora una aplicación que convierta este número a sus correspondiente valor cualitativo,


Uso de while

Solicitar un número positivo. Elabora una aplicación en PHP que imprima los números pares que hay entre 1 y ese número, incluyéndolo si es el caso.
